# 能量租赁机器人

### 功能
* 🔥 自助充值Telegram会员
* 🔥 支持动态配置支持平台/手动充值
* 🔥 支持余额功能

### 部署
* 本程序基于Telegram Bot，分为主程序telegram-premium
* 确保机器人部署服务可以访问外网`telegram.org`
* 使用自己的telegram生成一个机器人，并获取到token
* 更换`tron_scan_key`和`tron_grid_key`
* 配置文件`telegram-premium.yaml.example`改名为`telegram-premium.yaml`
